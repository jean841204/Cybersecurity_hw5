# 更新日誌

## [v1.1.0] - 2025-11-30

### 新增功能
1. **模型預載入**
   - 在頁面開啟時即載入模型，不需等待用戶點擊檢測按鈕
   - 提升使用者體驗，減少等待時間
   - 頁面頂部顯示模型就緒狀態

2. **模型資訊顯示**
   - 側邊欄新增"使用的模型"區塊
   - 可查看詳細的模型資訊：
     * 模型名稱：ChatGPT Detector RoBERTa
     * 模型類型：RoBERTa-base
     * 模型大小：約 500 MB
     * 訓練數據說明
     * 準確度資訊
     * 完整路徑

3. **判定原因說明**
   - 新增"判定原因與評比指標"區塊
   - 顯示具體的檢測指標
   - 解釋為什麼模型做出這樣的判定
   - 包含：
     * 檢測到的具體指標（強/明顯/輕微 AI/人類特徵）
     * 機率差距分析
     * 判定原因列表

4. **評比標準說明**
   - 新增可展開的"模型如何判定 AI 文本？"說明區
   - 詳細解釋 RoBERTa 模型的評比機制：
     * 語言模式分析
     * 上下文連貫性
     * 詞彙特徵
     * 寫作風格
   - 說明機率計算方法
   - 信心等級定義

### 優化改進
1. **性能提升**
   - 模型預載入避免重複載入
   - 檢測速度更快（首次點擊即可立即開始分析）

2. **用戶體驗**
   - 清晰的模型狀態顯示
   - 更詳細的判定理由
   - 透明的評比標準說明

3. **代碼結構**
   - 在 `model_loader.py` 中新增 `MODEL_INFO` 常量
   - 新增 `get_model_info()` 函數
   - `predict_ai_text()` 函數返回更多資訊：
     * reasons: 判定原因列表
     * indicators: 檢測指標列表
     * probability_difference: 機率差距

### 技術細節
- 使用 `@st.cache_resource` 確保模型只載入一次
- 新增 `initialize_app()` 函數處理應用初始化
- 頁面載入時自動調用模型載入
- 移除檢測按鈕中的重複模型載入邏輯
